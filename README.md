# 🩺 Pneumonia Detection using Deep Learning (CNN)

This project focuses on developing a deep learning-based medical image classification system to detect pneumonia from chest X-ray images using Convolutional Neural Networks (CNN).  
The system aims to assist early disease identification by automatically classifying lung X-ray images into Normal and Pneumonia categories.

---

## 🚀 Project Overview

Artificial Intelligence and deep learning techniques are widely used in medical image analysis to support diagnosis and treatment planning.  
In this project, a CNN model is trained on chest X-ray image datasets to learn important visual patterns related to pneumonia infection.  
The trained model can predict whether a given X-ray image represents a normal lung condition or pneumonia.

---

## 🎯 Objectives
